## README Screenshot Rule

Use only 3-4 screenshots in the README:

1. architecture diagram
2. Command Center overview
3. ArgoCD App of Apps
4. Grafana or Trino proof

Everything else belongs in this page.

---

## Avoid Publishing

Do not publish screenshots that show:

- passwords
- private tokens
- access keys
- secret values
- internal credentials
- unnecessary terminal clutter

Routes are okay if you are comfortable showing the lab domain, but credentials are not.
